## Sobre o Projeto
 
Análise exploratória do dataset [AI Workforce and Automation Dataset 2015–2025](https://www.kaggle.com/datasets/emirhanakku/ai-workforce-and-automation-dataset-20152025), que reúne indicadores econômicos e de mercado de trabalho de 20 países ao longo de 11 anos.
 
O objetivo é investigar como o avanço da IA e da automação se relaciona com variáveis de investimento, empregabilidade, produtividade e requalificação profissional.
 
## Estrutura do Repositório
 
```

├── global_ai_workforce_automation_2015_2025.csv            # Dataset utilizado
```

## Como Executar
 
1. Abra o notebook no [Google Colab](https://colab.research.google.com/github/matheusd09/mvp-analisedados-boaspraticas/blob/main/mvp_analisedados.ipynb).
2. Execute todas as células sequencialmente.
3. O dataset é carregado diretamente via URL deste repositório, sem necessidade de download manual.
